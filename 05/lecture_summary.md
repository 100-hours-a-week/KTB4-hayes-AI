### 모델 아키텍처

- 각 레이어 구성, 연결 방식, 활성화 함수, 입력 및 출력의 형태 등 모델의 전체 구조

### 사전 훈련 모델

- 이미 학습이 끝난 모델

### ResNet

- 출력과 입력의 차인 잔차를 학습하는 구조의 모델

### VGG16

- 16개 레이어로 구성된 CNN구조의 사전 학습 모델

### 파인 튜닝

- 사전 학습된 모델에 새로운 데이터셋으로 미세조정하여 학습시키는 과정

### Feature Extraction(특징추출)

- 사전 학습된 모델에 기존 층은 고정하고, 추출한 특징을 이용해 새로운 출력층만 학습하는 기법

### Zero-Shot

- 학습하지 않은 클래스에 대해서도 사전 지식을 바탕으로 예측을 하는 기법

### Overfitting

- 학습 데이터를 과하게 학습하여 실제 데이터를 제대로 예측하지 못하는 현상

### Underfitting

- 학습 데이터를 충분히 학습하지 못해 낮은 성능을 보이는 현상

### Hyperparameter Tuning

- 성능 최적화를 위해 모델 파라미터를 조정하는 과정

### NLP

- 자연어를 기계를 통해 묘사할 수 있게 연구하고 구현하는 분야

### RNN

- 순차적인 데이터를 처리하기 위한 모델

### LSTM

- 기존 RNN의 멀리 있는 정보는 소실된다는 문제를 해결하기위해 설계된 모델

### 원핫인코딩 vs 워드 임베딩

- 원핫인코딩: 총 단어 개수의 차원으로 표현하며 해당 단어의 값만 1이고 나머지는 0인 벡터
- 워드 임베딩: 사전에 정해진 차원으로 해당 단어를 표현하는 벡터

### 유사도 계산

- 벡터공간상에 두 텍스트가 얼마나 가까운 위치에 있는지 수학적으로 계산

### N-gram

- 앞에 등장한 n개의 단어로 다음 단어를 예측하는 모델

### 그래디언트 소실 (Vanishing Gradient)

- 기울기 값이 매우 작아져서 가중치가 업데이트 되지 않는 현상

### Hidden State (은닉 상태)

- 이전 시점까지의 입력정보를 요약해서 가지고 있고 다음 시점으로 전달되는 벡터
