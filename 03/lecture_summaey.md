
### Numpy

다양한 수학 연산을 할 수 있는 파이썬 라이브러리

### 차원

배열에서 축의 개수

### shape

배열의 차원별 요소 개수를 나타내는 튜플

### 데이터 타입

해당 요소가 가질 수 있는 값의 유형 ex) int, float, str 등등

### 인덱싱

배열에서 특정 값에 접근하는 방법

### 연산

수학적 또는 논리적 계산을 뜻한다.

### 유니버설 함수

배열의 모든 요소에 동일한 연산을 적용하는 함수 

### pandas

데이터를 더욱 다루기 쉽게하는 파이썬 라이브러리

### 시리즈(Series)

인덱스를 가지고 있는 1차원 배열

### 데이터프레임(DataFrame)

행과 열로 이루어져있는 2차원 배열

### 필터링(Filtering)

조건을 걸어서 원하는 값을 선택하는 과정

### 그룹화(Grouping)

기준을 정하여 데이터들을 그룹화하여 연산을 수행할 수 있게 하는 기능

### 병합(Merging)

여러 데이터프레임을 열이나 인덱스를 기준으로 결합하는 과정

### 결측치 처리

누락되거나 null인 값을 전처리하는 과정

### 피벗

데이터를 기준에 따라 재구성하고 요약해주는 기능

### 중복 제거

동일한 값을 가진 행을 제거하는 작업

### 문자열 처리

문자열이 포함된 데이터프레임에서 조작 및 변환하는 기능
