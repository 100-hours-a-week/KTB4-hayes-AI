### 데이터 전처리

- 데이터를 활용하기 좋은 상태로 정제하고 변환하는 과정

### 데이터 증강

- 기존 데이터를 변형하여 새로운 데이터를 생성

### 데이터셋 분할

- 데이터를 training, test, validation으로 분할하여 목적에 맞게 사용하는 것

### 머신러닝

- 기계가 데이터로 부터 학습하여 예측하거나 분류할 수 있는 것

### 랜덤 포레스트

- 여러 개의 의사결정트리를 결합하여 성능을 향상시키는 모델

### K-NN

- 데이터가 주어졌을 때 이 데이터와 가까운 이웃을 보고 예측하는 알고리즘

### SVM

- 주어진 데이터를 최적으로 분리하는 초평면을 찾는 알고리즘

### Naive Bayes

- 특성들이 서로 독립이라고 가정하고 베이즈 정리를 활용하여 분류하는 모델

### 딥 러닝

- 대량의 데이터로 스스로 학습하여 모델을 만드는 기술

### Perceptron

- 하나의 뉴런으로 여러 인풋을 받아 하나의 아웃풋으로 출력하는 함수

### Activation Function(활성화 함수)

- 뉴런의 출력을 결정하는 함수

### Artificial Neural Network

- 입력층, 은닉층, 출력층으로 구성되어 뇌의 뉴런 네트워크를 모방한 알고리즘

### Fully Connected Neural Network

- 모든 뉴런이 이전 층의 모든 뉴런과 연결된 신경망 구조

### Fully Connected Layer

- 모든 입력 뉴런이 모든 출력 뉴런과 연결된 레이어

### Loss Function

- 예측 값과 실제 값의 차이. 이 예측 오차를 최소화하기 위해 학습

### Backpropagation

- 예측 오차를 기반으로 가중치의 기울기를 계산하는 과정
