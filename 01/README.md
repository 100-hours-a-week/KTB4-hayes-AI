# 개인 비서 CLI 프로그램

터미널 환경에서 간단하게 개인 일정과 마감일을 관리할 수 있는 파이썬 기반의 CLI 프로그램입니다.

## 1. 주요 기능
* **일정 입력:** 사용자가 입력한 날짜, 제목, 상세 내용, 마감일을 로컬 환경에 `schedules.json` 파일로 자동 저장합니다.
* **조건별 일정 조회:** 오늘 일정, 내일 일정, 혹은 사용자가 지정한 특정 날짜의 일정을 필터링하여 조회합니다.
* **마감 임박 알림:** 조회 시 마감일이 2일 남은 일정을 자동으로 감지하여 경고 메시지를 표시합니다.
* **일정 선택 삭제:** 전체 저장된 일정 목록을 확인하고, 번호를 선택하여 원하는 일정을 삭제할 수 있습니다.
* **입력 환경 최적화:** `readline` 모듈을 활용하여 macOS 터미널 환경에서 발생하는 한글 입력 및 삭제(Backspace) 시의 커서 오작동 버그를 보정했습니다.

## 2. 개발 환경
* **Language:** Python
* **Database:** JSON (로컬 파일)
* **내장 모듈:** `json`, `os`, `readline`, `datetime`

## 3. 파일 구조
```text
01/
├── main.py            # 프로그램 메인 실행 파일
└── schedules.json     # 일정 데이터가 저장되는 JSON 파일 (.gitignore 추가)
```

## 회고
CLI 프로그램을 만들어 보는 것은 굉장히 오랜만이였던 것 같다. 
그래서인지 만들기가 뭔가 더욱 까다로웠고 기본적인 모듈 사용법이 하나도 기억이 안나서 전부 찾아가면서 작업을 했다.
굉장히 간단한 프로그램이지만 추후에 노션 api와 연동하고, 최종적으로 mcp 서버를 통해서 자연어로도 내가 원하는 일정을 간편하게 등록하는 프로그램을 계획하고 있다.
노션 api와 mcp 서버를 한번도 사용해 본 적이 없어서 배우면서 해야겠지만 내 계획대로 프로그램이 완성될 모습이 기대된다.
